# Cycled-Riemannian-Haar (CRH)

This repository will contain the official implementation of the **Cycled-Riemannian-Haar (CRH)** algorithm, A Wavelet Transform for SPD Matrix-Valued Processes with Perfect Reconstruction.

This code is part of the implementation for the paper:  
**"Cycled Riemannian Haar: A Wavelet Transform for SPD Matrix-Valued Processes with Perfect Reconstruction"**

## Code Release
The complete Python source code and installation instructions will be made publicly available in this repository immediately following the formal peer-review acceptance and publication of the paper.

For any immediate inquiries, please contact the Center for Complex Data Analysis (CCDA) at HIT.
